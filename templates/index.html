About This Project

Titanic Survival Predictor is a Machine Learning web application developed using Flask and Scikit-Learn. The model predicts whether a passenger would have survived based on details like age, gender, passenger class, fare, family size, cabin availability and more.

This project demonstrates Machine Learning model deployment, Feature Engineering, Flask Backend, HTML, CSS, Bootstrap and Responsive Web Design.
